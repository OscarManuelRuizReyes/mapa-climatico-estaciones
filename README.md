# Mapa interactivo de estaciones climáticas de México

Aplicación demostrativa en Streamlit para visualizar estaciones climatológicas seleccionadas en México y consultar precipitación y temperatura mensual/anual para el periodo 2015–2024.

La app utiliza datos procesados con fines demostrativos. No debe usarse como validación climatológica oficial.

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Despliegue

La app está preparada para desplegarse en Streamlit Community Cloud usando `app.py` como archivo principal.
